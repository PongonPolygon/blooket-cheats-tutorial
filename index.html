How to use Blooket Cheats

Regular Cheats

1. Go to your Blooket Cheats link and copy the code.

2. Open the lobby of the Blooket game. On the keyboard, hit Ctrl+Shift+I, or right click and click "inspect".

3. At the top of the Developer Tools menu, click "Console".

4. Paste the code into the console. If the "Allow Pasting" warning appears, type "allow pasting" and repaste the code to run the cheats.

Bookmark Cheats

1. Go to your Blooket Cheats link and copy the code.

2. Open your bookmark settings with Ctrl+Shift+O

3. Right click and click "Add new bookmark"

4. Name the cheats bookmark and paste the code. Remember: type "javascript:" before pasting in the code or it will not run.

5. Go to the lobby of the Blooket game and click on your bookmark.
